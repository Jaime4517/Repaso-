Publicación de la web

🚀 He utilizado GitHub Pages para publicar mi web.

📤 Subí mi proyecto a un repositorio de GitHub y luego activé GitHub Pages desde la configuración, seleccionando la rama principal.

🌍 GitHub es una plataforma que almacena proyectos de código y permite colaborar. GitHub Pages permite mostrar HTML/CSS públicamente en Internet.
