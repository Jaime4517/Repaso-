Conceptos básicos de HTML y CSS

HTML

  • <html> ... </html>: define el documento HTML
  • <head>: contiene información del documento
  • <body>: contenido visible
  • <h1> a <h6>: títulos de diferentes tamaños
  • <p>: párrafos de texto
  • <img>: imágenes
  • <a>: enlaces
  • <ul> y <li>: listas

CSS

🎨 CSS sirve para dar estilo a nuestras páginas: colores, fuentes, márgenes, tamaños, etc.

body {
    font-family: 'Roboto Mono', monospace;
    background-color: #121212;
    color: #f0f0f0;
}

h1 {
    color: #00ffff;
}
